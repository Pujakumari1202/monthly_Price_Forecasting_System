# Monthly Price Forecasting System

## Project Overview
The Monthly Price Forecasting System is a machine learning project developed to predict future monthly prices using historical price data. The project focuses on analyzing trends, patterns, and seasonal variations in the dataset to forecast prices for the next 12 months. The solution helps businesses make data-driven decisions for inventory management, budgeting, and strategic planning.

---

# Objective
The main objective of this project is to build a machine learning model that can accurately forecast future monthly prices based on historical data. The project aims to help businesses understand price trends and improve decision-making using predictive analytics.

---

# Problem Statement
Businesses often face difficulties in predicting future price fluctuations. Unexpected changes in prices can impact inventory planning, procurement, budgeting, and profitability. Manual forecasting methods are often inaccurate and time-consuming. This project addresses the problem by developing an intelligent machine learning-based forecasting system for predicting future monthly prices.

# Machine Learning Model
The project uses the **Random Forest Regressor** algorithm for price prediction because it provides:
- High accuracy
- Better handling of non-linear data
- Reduced overfitting
- Strong performance on tabular datasets

5. Model Building
6. Model Evaluation

# Evaluation Metrics
The model performance is evaluated using:
- Mean Absolute Error (MAE)
- Root Mean Squared Error (RMSE)
- R² Score
